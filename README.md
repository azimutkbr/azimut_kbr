# Сайт туристического клуба «Азимут»

Готовый статический сайт для GitHub Pages.

## Что внутри
- `index.html` — главная страница сайта.
- `style.css` — оформление сайта, светлая и тёмная темы.
- `script.js` — переключатель темы, мобильное меню, плавное появление блоков.
- `images/` — логотип и фотографии сайта.

## Публикация
Загрузите все файлы из этой папки в корень репозитория GitHub `azimut_kbr`, затем включите GitHub Pages: `Settings` → `Pages` → `Deploy from a branch` → `main` → `/root`.
